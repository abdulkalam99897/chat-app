Project Title: Real-Time Chat App (React Native + Node.js + MongoDB)

Developed By: M Abdul Kalam 



Project Description:
A real-time 1:1 chat app built using React Native (Expo) for frontend, Node.js with Express and Socket.IO for backend, and MongoDB Atlas for database.

Features:
• User registration and login (JWT based)
• Real-time messaging using Socket.IO
• Online/offline status indicators
• Message delivery confirmation
• Secure JWT authentication
• Logout and session management

Tech Stack:
Frontend: React Native (Expo)
Backend: Node.js + Express
Database: MongoDB Atlas
Socket Communication: Socket.IO

Setup Instructions:
1. Run server with `node index.js`
2. Run frontend with `npx expo start --tunnel`
3. Connect both using same local IP (update in Chat.js and Home.js)
4. Login and start chatting!
